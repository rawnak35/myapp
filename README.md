***Personal Finance Tracker (In Progress)***

A backend-focused Personal Finance Tracker built with Spring Boot, designed to help users track income, expenses, and balances over time. The project emphasizes clean architecture, data modeling, and production-ready backend practices, making it well-suited as a portfolio project for backend and data engineering roles.

*Features*

- Track income and expenses by category

- View current balance and historical transactions

- RESTful API design (CRUD operations)

- Input validation and error handling

- Layered architecture (Controller, Service, Repository)

- Ready for extension with analytics, reports, and authentication

*Tech Stack*

- Java 17

- Spring Boot (Web, Data JPA)

- Hibernate / JPA

- MySql

- Maven

- Docker (containerization)

- Microservices architecture (in progress / practice)**

*Architecture Overview*

The application follows a standard layered architecture:

- Controller layer – Exposes REST APIs

- Service layer – Business logic and validations

- Repository layer – Data access via Spring Data JPA

- Domain layer – Entities and enums

*Future Improvements*

- User authentication (JWT / OAuth2)

- Monthly and yearly reports

- Category-based analytics

- Export data to CSV

- deplayment
